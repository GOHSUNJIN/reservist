# Reservist Operations Portal

A web portal for military reservists to report, verify, and stay accounted for during shifts.

## Features

- **Check-in / Reporting** — Reservists log in and report for their shift
- **Verification** — Supervisors verify and track personnel accountability
- **Role-based access** — Separate views for Reservists and Admin/Supervisors
- **Account management** — Self-service registration and account deletion

## Stack

- Static HTML + JavaScript (no build step)
- `dc-runtime` declarative component framework via `support.js`
- IBM Plex Sans / IBM Plex Mono fonts

## Usage

Open `index.html` in a browser, or deploy as a static site (e.g. Vercel).

Use the **Quick Demo** buttons on the login screen to preview either the Reservist or Admin/Supervisor experience without creating an account.
